# Description

## Overview

This project serves as my portfolio, summarizing most of my important works in Game and VR Development till date. I have used github pages to host my website:

- https://rupamdebnath.github.io

Anyone can visit this link and view my work. The following frameworks/libraries/tools were used to create this project:
- Three.js
- Node.js
- vite
- github pages 
- github Actions

## Features

Various components of three.js has been used to create specific components to handle specific tasks

- PlayerCOntroller component to load and handle player movement
- An AudioHandler component to handle all types of audio
- A VideoHandler component to handle the functions for videos
- TextHandler component handles texts 
- A specific House class that creates the houses with textures and text
- Primitive objects available in three.js such as BoxGeometry, PlaneGeometry, ConeGeometry and SphereGeometry has been used to draw the house and bushes

# Instructions

Nothing specific, just open my websit at https://rupamdebnath.github.io and you should be good to go both on mobile and PC platforms, although I would suggest running it on PC for better performance.
On mobile, the touch control joystick can be used to control the movements of the fox, while on PC, the standard WASD and Shift for sprinting works.
